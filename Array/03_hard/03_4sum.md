# Problem: 4Sum
You are given an array of integers nums and an integer target. Your task is to find all unique quadruplets [nums[i], nums[j], nums[k], nums[l]] such that the indices i, j, k, and l are all distinct, and the sum of the four elements is equal to the target.

The solution set must not contain any duplicate quadruplets.

Example: Input: nums = [1, 0, -1, 0, -2, 2], target = 0 Output: [[-2, -1, 1, 2], [-2, 0, 0, 2], [-1, 0, 0, 1]]

## better(3 loops+hashmap)
````java
  class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        int n=nums.length;
        HashSet<List<Integer>> set=new HashSet<>();
        HashMap<Integer,Integer> map=new HashMap<>();
        for(int i=0;i<n;i++){
            for(int j=i+1;j<n;j++){
                map.clear();
                for(int k=j+1;k<n;k++){
                    long required=(long)target-((long)nums[i]+(long)nums[j]+(long)nums[k]);
                    if(required<Integer.MIN_VALUE || required>Integer.MAX_VALUE)continue;

                    if(map.containsKey((int)required))
                    {
                        List<Integer> temp=new ArrayList<>();
                        temp=Arrays.asList(nums[i],nums[j],nums[k],(int)required);
                        Collections.sort(temp);
                        set.add(temp);

                    }
                    map.put(nums[k],k);
                }
            }

        }
        List<List<Integer>> ans=new ArrayList<>();
        for(List<Integer> l:set)ans.add(l);
        return ans;
    }
}
````

## optimal(2 pointer )
````java
class Solution {
    /**
     * Finds all unique quadruplets in the array which sum up to the target,
     * using an optimized two-pointer approach with two fixed pointers.
     * @param nums The input array of integers.
     * @param target The target sum.
     * @return A list of unique quadruplets that sum to the target.
     */
    public List<List<Integer>> fourSum(int[] nums, int target) {
        int n = nums.length;
        // Sort the array to use the two-pointer technique and skip duplicates
        Arrays.sort(nums);
        List<List<Integer>> ans = new ArrayList<>();
        
        // Fix the first element (nums[i])
        for (int i = 0; i < n; i++) {
            // Skip duplicate 'i' elements
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            
            // Fix the second element (nums[j])
            for (int j = i + 1; j < n; j++) {
                // Skip duplicate 'j' elements (relative to 'i')
                if (j > i + 1 && nums[j] == nums[j - 1]) {
                    continue;
                }
                
                // Now, use two pointers 'k' and 'l' for the remaining part
                int k = j + 1;
                int l = n - 1;
                
                while (k < l) {
                    // Use 'long' for the sum to prevent integer overflow
                    long sum = (long) nums[i] + nums[j] + nums[k] + nums[l];
                    
                    if (sum < target) {
                        // Sum is too small, move the left pointer 'k'
                        k++;
                    } else if (sum > target) {
                        // Sum is too large, move the right pointer 'l'
                        l--;
                    } else {
                        // Found a quadruplet that sums to the target
                        List<Integer> temp = Arrays.asList(nums[i], nums[j], nums[k], nums[l]);
                        ans.add(temp);
                        
                        // Move both pointers
                        k++;
                        l--;
                        
                        // Skip duplicate 'k' and 'l' elements
                        while (k < l && nums[k] == nums[k - 1]) {
                            k++;
                        }
                        while (k < l && nums[l] == nums[l + 1]) {
                            l--;
                        }
                    }
                }
            }
        }
        return ans;

        // Time Complexity: O(N^3)
        // - Sorting takes O(N log N).
        // - The outer 'i' loop runs O(N) times.
        // - The inner 'j' loop runs O(N) times for each 'i'.
        // - The 'while' loop (two-pointer scan) runs O(N) times for each 'i' and 'j'.
        // - Total time = O(N log N) + O(N * N * N) = O(N^3).

        // Space Complexity: O(log N) or O(N) + O(M)
        // - O(log N) or O(N) for the sorting algorithm's internal space.
        // - 'M' is the number of unique quadruplets found, used to store the 'ans' list.
    }
}
````
